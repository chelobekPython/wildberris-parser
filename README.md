# How to use
# Linux (ubuntu)
1. Copy the URL of repository and print in terminal git clone https://github.com/chelobekPython/wildberris-parser. Then open popup.js and paste your tg bot token and your own chat ID in "YOUR_BOT TOKEN" and "YOUR_CHAT_ID".
2. Open google chrome and go to chrome://extensions/. Turn on developer mode and click on "upload the unpacked extension"
3. Select folder of parser and turn it on.
# Ready!
Now you can open wildberris.ru and parse it!

Happy parsing!
