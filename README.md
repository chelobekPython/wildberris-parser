# How to use 
1. You have to dowload zip archive on top of this page and extract it. Then open popup.js and paste your tg bot token and your own chat ID in "YOUR_BOT TOKEN" and "YOUR_CHAT_ID".
2. Open google chrome and go to chrome://extensions/ . Turn on developer mode and click to "upload the unpacked extension"
3. Select extracted folder and turn it on.
# Ready!
Now you can open wildberris.ru and parse it!

Happy parsing!
